Welcome

This website is made for a housing society. It helps residents register, submit survey details, and view information about representative elections.

The project is created using only HTML and CSS. It contains separate pages for login, registration, survey, election, results, and contact details.

Main Features

  • Resident registration page
  • Login page design
  • Society survey form
  • Representative election page
  • Sample result table
