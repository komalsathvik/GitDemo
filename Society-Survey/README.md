# Society Survey and Representative Election Management System

This is a medium basic HTML and CSS project. It uses multiple HTML pages and one common CSS file.

## Pages

- `index.html` - home page
- `login.html` - login form
- `register.html` - resident registration form
- `survey.html` - society survey form
- `election.html` - candidate list and voting form
- `results.html` - sample survey and election results
- `contact.html` - society contact and feedback form

## Note

This project uses only HTML and CSS.
